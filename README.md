# Calculator App

A simple calculator application built with C# using Visual Studio.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- User-friendly interface

## Technologies Used
- C#
- .NET
- Visual Studio

## How to Run
1. Clone the repository
2. Open the solution file in Visual Studio
3. Build and run the project
